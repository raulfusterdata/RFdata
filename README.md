# RFData - Simulador de Optimización Financiera 📊

Aplicación web interactiva desarrollada en Python (Streamlit) para el análisis de finanzas personales y proyección patrimonial mediante simulaciones estadísticas.

## Módulos de la herramienta:
1. **Test de Salud Financiera:** Diagnóstico automatizado basado en la regla 50/30/20.
2. **El Ladrón Invisible:** Cálculo del impacto de la inflación frente al coste de inacción.
3. **Simulación de Montecarlo:** Proyección probabilística de inversiones a largo plazo evaluando 200 escenarios macroeconómicos posibles.
4. **Plan de Acción:** Sistematización del ahorro y reglas psicológicas de inversión.

## Tecnologías empleadas:
* Frontend & Framework: `Streamlit`
* Cálculo matricial y estadístico: `NumPy`, `Pandas`
* Visualización de datos: `Plotly Graph Objects`
